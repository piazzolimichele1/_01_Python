# Python

This is the Python portfolio behind the trading pipeline I run top to bottom on the markets, every day.

This repository holds the Python side of an automated CFD trading operation on MT5:
the market data, the live strategies, and the applications that keep it all running.

## Sections
- **_01_Json-MarketData** : MT5 price history in JSON, the data the strategies are built on.
- **_02_LiveStrategies** : 109 live strategies across NAS100, US30 and SP500, each on its own Python engine.
- **_03_PythonApps** : the applications that helps me to run and monitor the pipeline, Python Monitor, Trade Updater, Trade Journal and GSheet Control.

Each section carries its full content in the Releases.
